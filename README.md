# Algorithmic Turn in HRM — Bibliometric Analysis (2018–2026)

## About
This project is a bibliometric study analyzing how Artificial Intelligence and algorithmic systems have transformed Human Resource Management (HRM) research over 2018–2026. It analyzes 6,546 Scopus-indexed publications and introduces the Algorithmic Transformation Index (ATI) to measure this shift.

## Key Findings
- Technology-oriented HRM discourse grew by 47.15%
- Algorithm-related terms grew by 4,183.87%
- ATI increased from 0.931 to 1.343 (pre-AI to post-AI period)
- Traditional HRM themes remained stable — showing a layering effect, not replacement

## Dataset
- Source: Scopus Database
- Period: 2018–2026
- Total Publications: 6,546

## Tools Used
- Python (pandas, scipy, matplotlib, seaborn, nltk)
- Scopus Database
- Jupyter Notebook

## How to Run
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Add raw Scopus data to data/raw/
4. Run notebooks in order (01 to 05)
